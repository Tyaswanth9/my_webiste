Overview

Crowdfunding is a way of raising money where many people contribute small amounts to support a project, idea, or business, usually done online.

Types of Crowdfunding:

  • Reward-based: Backers receive rewards.
  • Equity-based: Backers get shares or equity.
  • Donation-based: No return expected.
  • Loan-based: Money is repaid with interest.

Tools Used

  • Microsoft Excel
  • Power BI
  • Tableau
  • MySQL

Data & Cleaning Process

Data Sources: 4 Excel files: category, location, creator, project

Data Cleaning Steps (Power Query):

  • Imported all files
  • Removed duplicates
  • Sorted and cleaned columns
  • Formatted dates and text properly

Data Modeling

Created relationships between datasets to prepare a data model for analysis.

Key Insights

  • Total Projects: 366,000
  • Total Backers: 366,000
  • Total Amount Raised: $16,041M
  • Average Project Duration: 79 days
  • Projects by Outcome: 169

Excel Dashboard

  • Created calendar table using date/time functions
  • Applied formulas like SUM, COUNT, AVERAGE, SUMIF, IF
  • Built KPIs and charts for analysis

Power BI Dashboard

  • Imported Excel files into Power BI using Power Query
  • Used DAX functions: COUNT, SUM, DIVIDE, etc.
  • Converted epoch dates to readable dates
  • Created calculated columns and custom KPIs
  • Designed interactive visuals and dashboards

Tableau Dashboard

  • Converted CSV files to Excel
  • Imported all data into Tableau
  • Built relationships between data sources
  • Created measures using LOD expressions
  • Developed interactive dashboards

SQL Analysis (MySQL)

Functions Used: FROM_UNIXTIME(), DATE(), YEAR(), MONTHNAME(), QUARTER(), WEEKDAY(), DAYNAME(), CASE, CONCAT(), COUNT(), DISTINCT, SUM(), ROUND(), AVG(), DATEDIFF(), GROUP BY, ORDER BY, WHERE, subqueries

Sample Queries:

  • Total number of projects: SELECT COUNT(projectid) AS total_projects FROM projects;
  • Projects per year: SELECT YEAR(FROM_UNIXTIME(created_at)) AS year, COUNT(*) AS total FROM projects GROUP BY year;
  • Projects per month: SELECT MONTH(FROM_UNIXTIME(created_at)) AS month, COUNT(*) AS total FROM projects GROUP BY month;
  • Epoch dates converted to human-readable and calendar table creation included

Conclusion

  • High percentage of projects were successful overall
  • 2015 had the highest total amount raised
  • Average duration of projects reduced in 2015
  • Film & Video top successful category every year
  • Design was second-most successful
  • Failed projects slightly outnumbered successful ones in most years

  • Tableau: LOD expressions, data connections, dashboards
  • MySQL: Querying, filtering, data aggregation

This project helped me understand end-to-end data handling and dashboard creation.
