NIRF College Ranking Dashboard Project

Overview

NIRF stands for National Institutional Ranking Framework, developed by the Indian Government to rank colleges and universities in India.It was launched in 2015 to improve transparency and assess institutional performance.

Ranking Parameters :

  • Teaching, Learning and Resources
  • Research and Professional Practices
  • Graduation Outcomes
  • Outreach and Inclusivity

Business Problem

To analyze NIRF data and build dashboards that help understand:

  • Institutional performance
  • Average scores and rankings
  • Insights by state and grade
  • Data-driven decision-making

Data Sources

  • institute_details.csv
  • institute_location.csv
  • institute_ranking.csv

SQL

  1. View Creation: CREATE VIEW university_ranking AS ...
  2. KPI Tracking: SELECT COUNT(institute_id) AS "Total institutions", AVG(score) ...
  3. Top 5 Ranked Institutions: SELECT rank_, name FROM university_ranking WHERE rank_ < 6;
  4. State-wise Average Score: SELECT state, ROUND(AVG(score), 2) ...
  5. Grade-wise Institutions: SELECT CASE WHEN score >= 90 THEN 'A+' ...
  6. State-wise Institution Count: SELECT state, COUNT(institute_id) ...
